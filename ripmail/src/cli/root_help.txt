Commands:

Setup and management
  ripmail setup [--email <e>] [--password <p> | --google-oauth] [--openai-key <k>] [--id <mailbox_id>] [--imap-host …] [--imap-port …] [--no-validate] [--no-skill]
  ripmail config [--id <mailbox_id>] [--email <addr>] [--preferred-name …] [--mailbox-management on|off]  # non-secret settings (identity, IMAP archive behavior)
  ripmail sources list|add|edit|remove|status   # config.json sources[] (IMAP, Apple Mail, local directory, calendar); see ripmail sources add --help
  ripmail skill install                              # copy embedded `/ripmail` agent skill (Claude Code; OpenClaw if ~/.openclaw/skills exists)
  ripmail wizard [--gmail] [--no-validate] [--clean] [--yes]   # --gmail: Gmail sign-in in browser
  ripmail clean [--yes]   # wipe out all ripmail data and config (if --yes)
  ripmail backfill [<spec>] [--since <spec>] [--foreground] [--source <email|id>]  # historical mail (long-running; --foreground waits; else background + log path)
  ripmail rebuild-index

Monitor & manage inbox
  ripmail refresh [--force] [--text] [--source <email|id>]  # incremental sync (new mail); run backfill first when the index is empty or you need older history
  ripmail ask "<question>" [--verbose]
  ripmail inbox [<window>] [--since YYYY-MM-DD] [--thorough] [--text]
  ripmail rules --help                             # manage mail classification rules to improve signal to noise
  ripmail archive <message_id> … [--undo] [--source <email|id>]

Search & read
  ripmail search <query> [flags]
  ripmail who [query] [flags]
  ripmail whoami [--source <email|id>] [--text]
  ripmail read <TARGET>… [--source <id>] [--raw] [--json] [--text]   # TARGET = file path or Message-ID
  ripmail calendar list-calendars|today|upcoming|search|read  (see ripmail calendar --help; use -S/--source to scope)
  ripmail thread <id> [--json] [--text]
  ripmail attachment list <message_id>
  ripmail attachment read <message_id> <index>|<filename>

Sending mail
  ripmail send [--to ... --subject ...] [<draft-id>] [--source <email|id>]
  ripmail draft list|view|new|reply|forward|edit|rewrite  (see ripmail draft --help; draft new --source for a non-default account; prefer new|reply|forward --instruction and draft edit over raw --body)

Other
  ripmail status [--json]
  ripmail stats [--json]

Run ripmail <command> --help for command-specific options.

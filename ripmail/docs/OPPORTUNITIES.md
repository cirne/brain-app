# Product Opportunities

Improvement ideas discovered through real usage. Each entry captures the problem, a concrete example, and a proposed direction.

See [VISION.md](./VISION.md) for product vision, [ARCHITECTURE.md](./ARCHITECTURE.md) for technical decisions.

**Monorepo (brain-app):** This file indexes **ripmail-only** opportunities under [`opportunities/`](opportunities/) and [`opportunities/archive/`](opportunities/archive/). The parent **brain-app** list (chat, wiki, Tauri, hosted staging, …) lives at **[`../../docs/OPPORTUNITIES.md`](../../docs/OPPORTUNITIES.md)**. **`OPP-*` IDs are not shared** between ripmail and brain-app; cross-link when work spans both (e.g. OAuth relay, unified sources). Repository-wide context: **[`../AGENTS.md`](../AGENTS.md)**.

**Strategic sequencing:** Send (draft + SMTP) is **implemented in code** ([OPP-011 archived](opportunities/archive/OPP-011-send-email.md), [ADR-024](ARCHITECTURE.md#adr-024-outbound-email--smtp-send-as-user--local-drafts)); product may still sequence broad promotion behind read/sync validation. See that opp for **remaining work**.

**MCP deferral (2026-04-10):** Former **OPP-040** (gating / benefit bar for in-tree MCP) is **merged into** [OPP-039](opportunities/OPP-039-mcp-deferred-cli-first.md).

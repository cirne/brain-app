# Note template

Use for **`notes/<slug>.md`** or dated captures. **Ephemeral by default** — fold facts into durable pages, then delete or stub this file.

## Context

- **When / why** this note exists *(date, meeting, trigger — optional but helps stale detection).*

## Capture

- Short bullets — **not** a full wiki page unless it graduates to `topics/` or `projects/`.

## Where facts live long-term

- **Link** the canonical home: **`[[people/…]]`**, **`[[projects/…]]`**, **`[[topics/…]]`** — after merging, trim or remove this note.

## Cleanup

- When superseded: **delete** this file or replace with one line pointing to the durable page.

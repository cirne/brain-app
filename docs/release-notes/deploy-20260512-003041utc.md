# Braintunnel release notes
**Release date:** 2026-05-12
**Changes since:** deploy-20260512-001349utc

This release improves the visibility and accuracy of Gmail backfill status in the Hub, making it easier to track sync progress.

---

## What's new

- **Enhanced Gmail backfill status** — The Hub now shows Gmail backfill as live with accurate progress using the listed-ID target.
- **Improved sync summary data** — Persisted historical messages.list totals for better tracking and exposed backfillListedTarget in multiple API payloads.
- **User interface updates** — Added backfill banner and clearer indexed vs listed target copy with internationalization support.

Also in this release: minor fixes and deployment tooling improvements to keep things running smoothly.

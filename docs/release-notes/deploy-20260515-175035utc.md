# Braintunnel release notes
**Release date:** 2026-05-15
**Changes since:** deploy-20260515-173052utc

This release focuses on improving email synchronization reliability by ensuring proper backfill behavior for password-protected IMAP accounts.

---

## What's new

- **Improved IMAP sync reliability** — The system now honors Hub backfill settings for password IMAP accounts, ensuring historical emails are properly fetched from the start.
- **Enhanced backfill logic** — Sync state is cleared to trigger refetching from UID 1, improving completeness of email syncs.
- **Documentation updates** — Clarified semantics on IMAP types to aid developer understanding.

Also in this release: minor build and deployment process improvements.

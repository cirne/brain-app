# Braintunnel release notes
**Release date:** 2026-05-09
**Changes since:** deploy-20260508-191207utc

This release brings a major update to settings navigation, onboarding experience, and ripmail synchronization with enhanced observability and diagnostics. We've also improved agent chat features and tightened HTTP sync reliability.

---

## What's new

- **Settings and Hub navigation improvements** — Added new Settings subpages for Connections and Wiki, plus workspace drill-downs; hub overview now opens Connections automatically when relevant.
- **Onboarding enhancements** — Split onboarding UI for clarity, integrated initial bootstrap agent, improved mail coverage prompts, and streamlined onboarding API paths.
- **Ripmail sync and observability** — Introduced scheduled tenant-aware mail sync with OTLP tracing support, enhanced diagnostic logs, and hardened Gmail API HTTP calls with timeouts.
- **Agent and chat updates** — Improved SSE streaming, added conversation finish shortcuts, and expanded diagnostics for smoother interaction.
- **Brain-query improvements** — Normalized rejection reasons for too broad queries and refined system prompts to favor research tools and wiki guidance.

Also in this release: updated New Relic custom attributes and docs, cleaned up Enron tooling, added unit tests for metadata parsing, and various bug fixes and polish.

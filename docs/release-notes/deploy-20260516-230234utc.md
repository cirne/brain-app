# Braintunnel release notes
**Release date:** 2026-05-16
**Changes since:** deploy-20260516-182019utc

This release advances our Slack integration with venue-aware routing and inbound DM support, fixes attachment storage collisions, and improves calendar queries and voice input alignment.

---

## What's new

- **Slack venue-aware routing:** Slack messages now route intelligently by venue type (DM, channel, private group), with bot user ID resolution and improved mention parsing.
- **Inbound Slack DM integration:** Full inbound Slack DM pipeline added, including turn runner, approval flow, and Slack interactions endpoint for seamless two-step approve/reject.
- **Slack workspace OAuth and user linking:** Ship Slack install and user link flows, global connection store, and platform-agnostic messaging adapter with UI support.
- **Attachment storage fix:** Prevented UID collisions in Ripmail attachments by storing files in per-message subdirectories.
- **Calendar query improvements:** Calendar day queries now respect Hub default calendars instead of only primary, with clearer guidance and empty-result tooltips.
- **Voice composer UI alignment:** Voice mode input shell now matches keyboard input layout for consistency and better user experience.
- **Slack approval sync:** Brain UI approve/decline actions now sync back to Slack Block Kit messages, keeping Slack and Brain UI in sync.

Also in this release: minor bug fixes and polish to improve stability and user experience.

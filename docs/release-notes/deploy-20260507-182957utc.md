# Braintunnel release notes
**Release date:** 2026-05-07
**Changes since:** deploy-20260506-202609utc

This release brings significant improvements to email and calendar synchronization, enhanced assistant capabilities, and important architectural documentation for cloud tenant lifecycle management.

---

## What's new

- **Enhanced Email Sync and Ripmail Improvements** — Introduced Gmail REST incremental refresh with history tracking, improved Google Calendar sync with targeted database operations, and hardened ripmail rebuild and CLI testing.
- **Assistant Updates** — Revised the assistant base prompt for clearer principles-first guidance and added new skills for ripmail-cli and sync log inspection.
- **Cloud Tenant Lifecycle Architecture** — Added comprehensive documentation detailing the S3-backed tenant lifecycle, distributed locking, local hot storage, and backup/restore workflows to support large-scale hosted deployments.

Also in this release: improvements to CI workflows, test coverage expansions, UI enhancements like slide-over headers and token meters, and various minor bug fixes and polish.

# Braintunnel release notes
**Release date:** 2026-05-12
**Changes since:** deploy-20260512-003041utc

This release brings enhanced visual artifact support for mail attachments, improved inbox and dock navigation, and smoother shell routing with connection status feedback.

---

## What's new

- **Visual artifacts for mail and files** — Preview cards, image viewers, and tool block rendering now enhance your mail and indexed file experience with thumbnail previews and streaming hooks.
- **Inbox dock compose and navigation helpers** — Compose and navigation in the inbox dock are streamlined with new helpers supporting email and wiki docks, improving workflow efficiency.
- **Shell routing and connection UX improvements** — Updated primary surface routing, connection status overlays, and assistant rail components provide clearer navigation and connection feedback.

Also in this release: documentation updates for Tailwind spacing tokens and various test and UI polish improvements.

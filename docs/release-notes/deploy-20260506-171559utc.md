# Braintunnel release notes
**Release date:** 2026-05-06
**Changes since:** deploy-20260505-214005utc

This release brings major improvements to mail sync performance, onboarding experience, and wiki usability. We've enhanced parallel UID FETCH in ripmail for faster mail ingestion, introduced phased and background onboarding orchestration, and polished the wiki UI with collapsible breadcrumbs.

---

## What's new

- **Faster mail sync with parallel UID FETCH** — Ripmail now supports pipeline parallelism with tunable sync knobs, enabling efficient batch fetching and ordered checkpoint ingest.
- **Improved onboarding orchestration** — Background task queue, onboarding state machine, and phased ripmail sync streamline first-run setup and mail backfill.
- **Wiki UI enhancements** — Collapsible breadcrumbs in the primary bar improve navigation on narrow screens.
- **Onboarding and mail UI updates** — Extended API status hints, profiling with auto-advance retry, and hub activity overview for better user feedback.

Also in this release: enhanced server tests, documentation updates, and bug fixes including addressing BUG-044 for wiki breadcrumbs.

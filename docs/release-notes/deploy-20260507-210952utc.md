# Braintunnel release notes

**Release date:** 2026-05-07

**Changes since:** deploy-20260507-202553utc

This release includes an important fix to the slide header cells to prevent effect depth loops, improving stability and performance.

---

## What's new

- **Improved slide header cells** — Updated slide header cells to use claim/patch mechanisms instead of register/updateSeq, preventing effect depth loops and enhancing reliability.
- **Controller patch API introduced** — Added a new patch API for controllers to streamline updates and state management.
- **SlideOver migration** — Migrated SlideOver consumers and contexts to the new owner-token cell system, improving consistency and maintainability.
- **Regression tests added** — Implemented regression tests to catch and prevent recurrence of the slide header cells issue.

Also in this release: Documentation updates clarifying failure modes related to slide header cells and controller patching.

# Braintunnel release notes
**Release date:** 2026-05-11
**Changes since:** deploy-20260511-145605utc

This release enhances email parsing, calendar integrations, and agent interactions with improved concurrency and resilience.

---

## What's new

- **Enhanced email parsing and indexing** — Sender-aware From fields, normalized attachments, and draft guards improve mail handling and reply/forward reliability.
- **Robust maildir management** — Namespace raw .eml files by UIDVALIDITY, clear folders on UIDVALIDITY changes, and add regression tests to prevent data drift.
- **Improved inbox HTML rendering** — Validate raw .eml Message-ID against DB rows to ensure consistent HTML display and fallback to stored text when needed.
- **Streaming agent improvements** — Better markdown parsing, wiki tools, and HTML rendering enhance chat UI and agent workflows.
- **Concurrent source refresh** — Bounded concurrency for source refreshes with inbox summary returns and query reply notifications for smoother sync.
- **Google Calendar API integration** — Source-aware event cancel/delete mutations now call Google APIs, with recurrence handling and sync client UI.
- **OAuth and feedback system hardening** — Automatic OAuth cleanup on invalid_grant errors, improved calendar and feedback UI, and onboarding copy updates.

Also in this release: bug fixes, test coverage expansions, documentation updates, and UI polish across multiple components.

# Braintunnel release notes

**Release date:** 2026-05-05

**Changes since:** deploy-20260505-165221utc

This release brings enhanced wiki sharing capabilities, improved OAuth calendar integrations, and multiple bug fixes across the wiki, Ripmail, and client navigation.

---

## What's new

- **Enhanced Wiki Sharing** — Grantee handles now appear in the wiki share API and UI, improving clarity on share recipients.
- **TipTap Editor Improvements** — Added Escape key dismissal for the floating block menu, streamlining editor interactions.
- **OAuth Calendar Fixes** — Google Calendar calendarIds are preserved across Gmail OAuth callbacks, preventing calendar selection loss.
- **Bug Fixes and Stability** — Addressed Ripmail archive issues, wiki overlay and navigation bugs, and improved client breadcrumb and SlideOver behaviors.

Also in this release: updated tests for floating menu and OAuth calendar handling, documentation updates, and deployment note enhancements.

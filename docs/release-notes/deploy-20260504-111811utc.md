# Braintunnel release notes

**Release date:** 2026-05-04

**Changes since:** deploy-20260503-213300utc

This release focuses on improving the ChatHistory rail UI with a modernized styling approach and subtle visual enhancements to the chat interface.

---

## What's new

- **UI modernization** — Migrated the ChatHistory rail from scoped CSS to Tailwind CSS utilities, improving consistency and maintainability.
- **Visual polish** — Updated chat rail primary actions and hint chips to use transparent styling for a cleaner look.

Also in this release: minor CSS cleanup and alignment of touch hover policies for a smoother user experience.

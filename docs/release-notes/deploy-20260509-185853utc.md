# Braintunnel release notes
**Release date:** 2026-05-09
**Changes since:** deploy-20260509-182210utc

This release improves the onboarding experience and developer tooling with enhanced app configuration and reset capabilities.

---

## What's new

- **Improved onboarding flow** — Users can now exit onboarding via interactive chips instead of automatic conversation finishes, allowing smoother and more natural sign-offs.
- **New app configuration API** — Added `/api/config` endpoint for managing app settings programmatically.
- **Developer soft reset feature** — Introduced a POST `/api/dev/soft-reset` endpoint to clear client storage and reset state without a full reload.
- **Wiki-first onboarding prompts** — Updated onboarding interview and bootstrap prompts to prioritize wiki pages, person-finding, and web search tasks.
- **Access control tightened** — Web search access is now limited to onboarding interviews, and vault/onboarding routes have stricter permissions.

Also in this release: updated tests for new APIs and onboarding prompts, plus various minor fixes and polish.

# Braintunnel release notes

**Release date:** 2026-05-15

**Changes since:** deploy-20260514-201545utc

This release brings significant improvements to our B2B filtering, email and document handling, and search capabilities. We've enhanced evaluation tooling, refined mail body extraction, and improved integration with indexed documents and spreadsheets.

---

## What's new

- **Enhanced B2B filtering and evaluation** — baked the always-omit baseline into outbound filters, expanded payment-rail identifier rules, and refactored evals to run across multiple privacy policies with improved JSONL orchestration.
- **Improved mail reading experience** — mail bodies that are HTML-only now surface properly to agents using consistent markdown conversion, ensuring no content is missed.
- **Integrated indexed documents and spreadsheets in chat** — agents now see inline indexed viewer text and multi-sheet spreadsheets merged into chat context, supporting native XLSX exports and better spreadsheet tab navigation.
- **Search and Drive integration enhancements** — added URL resolution for indexed entries, content fingerprint validation for Drive caches, and improved search guidance with zero-hit and broad-hit hints.

Also in this release: backfilled mail status date bounds for consistency, fixed zero-byte Drive file fetches, aligned commit skill workflows with sync and publish steps, and expanded test coverage for mail and file search.

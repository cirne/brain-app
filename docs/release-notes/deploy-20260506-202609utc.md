# Braintunnel release notes

**Release date:** 2026-05-06

**Changes since:** deploy-20260506-171559utc

This release focuses on improving the mobile editing experience and polishing the BrainHub interface, alongside some backend cleanup and documentation updates.

---

## What's new

- **Mobile editing improvements** — Fixed TipTap editor's floating menu behavior on mobile, resolving empty-editor issues and enhancing block menu visibility and escape key handling.
- **BrainHub enhancements** — Added BrainHubPage, HubSourceRowBody, and HubActivityOverview components with tests for a smoother hub experience.
- **Unified chat and wiki updates** — Minor wiring improvements to AgentInput, UnifiedChatComposer, and Wiki components for better integration.
- **Backend cleanup** — Simplified onboarding routes and ripmailHeavySpawn logic to streamline server operations.

Also in this release: updated documentation archive and release notes for clarity and completeness.

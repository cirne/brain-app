# Braintunnel release notes
**Release date:** 2026-05-08
**Changes since:** deploy-20260507-222122utc

This release introduces major advancements in Brain Query capabilities, Brain Access management, and agent LLM configuration, along with UI improvements and policy enhancements.

---

## What's new

- **Brain Query delegation and rejection tools** — Added a new reject_question tool for early rejection handling, integrated brain query API routes, and enhanced client UI with answer previews and policy breadcrumbs.
- **Brain Access hub and policy management** — Introduced a comprehensive Brain Access UI including policy hubs, grants, outbound grants, and policy templates; removed wiki share routes to streamline access control.
- **B2B feature gating and refined access UI** — Brain Query features are now gated behind a B2B feature flag with improved policy flows, activity details, and grant filtering.
- **Agent LLM configuration enhancements** — Added support for configuring agent LLMs via BRAIN_LLM and BRAIN_FAST_LLM environment variables, enabling fast tier models for suggestions and repairs.
- **UI and documentation polish** — Fixed padding issues on brain-access pages for narrow viewports; updated docs with architecture drafts and cross-links for Brain Query and access policies.

Also in this release: Enron demo tenant seed data and e2e helpers, Docker and docs updates, onboarding mail gate wiring, and various test improvements for B2B and agent tool registry.

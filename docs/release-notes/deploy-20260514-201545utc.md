# Braintunnel release notes
**Release date:** 2026-05-14
**Changes since:** deploy-20260514-175812utc

This release introduces enhanced multi-source search capabilities and Google Drive integration improvements, streamlining your document and email workflows.

---

## What's new

- **Multi-source search:** Emails can now be resolved to both IMAP and Google Drive source IDs, enabling more comprehensive search results.
- **Scoped Drive reads:** The read_indexed_file function now filters document indexes by source ID for more precise content retrieval.
- **Google Drive file sync:** Added a Drive sync pipeline with cached reads and concurrency limits to improve performance and reliability.
- **Simplified Drive UI:** Removed the hubDriveSuggest API and streamlined the Drive connector UI and internationalization for easier use.
- **Documentation updates:** Architecture docs and integration indexes have been aligned with the new external source syncing direction, including updated wiki vs mail read paths.

Also in this release: policy selection parity improvements across browsers and various minor fixes and polish.

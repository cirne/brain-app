# Braintunnel release notes

**Release date:** 2026-05-03

**Changes since:** deploy-20260503-152101utc

This release brings improvements to onboarding visuals and a major update to wiki sharing with more flexible invite options and database schema management.

---

## What's new

- **Enhanced onboarding UI** — Circular accents added to onboarding steps and a new wiki pulse dot for better visual cues.
- **Wiki sharing improvements** — Invites can now be sent by grantee ID, handle, or email, improving flexibility and accuracy.
- **Global database schema management** — Added brain_global_schema version tracking and automatic rebuild of the global DB on schema changes to ensure data integrity.
- **API enhancements for wiki shares** — The POST /api/wiki-shares endpoint now supports multiple grantee identification methods and keys invites to tenant IDs instead of mailboxes.

Also in this release: token meter SVG clipping refined, spinner CSS indentation tightened, and updated documentation for wiki sharing and release notes.

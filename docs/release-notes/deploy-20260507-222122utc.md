# Braintunnel release notes
**Release date:** 2026-05-07
**Changes since:** deploy-20260507-210952utc

This release brings UI polish to chat transcripts, a unified and improved wiki breadcrumb navigation, and important bug fixes for slide headers and wiki path handling.

---

## What's new

- **Improved chat transcript UI** — Added padding inside the scrollport for a cleaner chat transcript display and ensured consistent scrollbar placement.
- **Unified wiki breadcrumb navigation** — Refactored breadcrumb logic so the primary bar and slide-over share the same crumbs, supporting personal, vault-relative, and shared paths.
- **Wiki path cleanup** — Automatically collapse repeated `me/` prefixes in agent wiki paths to prevent clutter and confusion.
- **Slide header bug fix** — Fixed a depth loop issue in slide header effects for smoother overlay and pane interactions.

Also in this release: regression tests for wiki path fixes, starter wiki template updates, and documentation improvements including bug archival and prompt updates.

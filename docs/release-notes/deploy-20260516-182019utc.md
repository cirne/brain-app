# Braintunnel release notes
**Release date:** 2026-05-16
**Changes since:** deploy-20260515-203527utc

This release introduces exciting Slack integration features, significant backend refactoring, and improvements to mail handling and developer workflows. We've also enhanced privacy policy management and upgraded core tooling for a smoother development experience.

---

## What's new

- **Slack integration** — Added a hello-world Events API endpoint with always-on brain-to-brain UI and webhook testing support.
- **Policy and privacy enhancements** — Introduced integration-scoped policy models and outbound deny fences for privacy presets to improve data handling.
- **Wiki and onboarding refactor** — Consolidated bootstrap logic into a single expansion runner simplifying the build lifecycle and UI onboarding states.
- **Mail improvements** — Throttled Gmail backfill with better concurrency control and fixed draft forwarding to include original quotes and recipient resolution.
- **Developer experience upgrades** — Migrated to pnpm v11, added portless URLs for dev environments, and improved git worktree bootstrap workflows.

Also in this release: cached Enron mail tarball outside data folders, refined chat transcript tool-call summaries, and removed deprecated inbox draft edit routes for cleaner API surface.

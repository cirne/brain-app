# Braintunnel release notes
**Release date:** 2026-05-11
**Changes since:** deploy-20260509-185853utc

This release delivers a major upgrade to our mail handling with a full TypeScript in-process ripmail port, enhanced collaboration tools, tenant SQLite support, and a phased rollout of i18n across the UI.

---

## What's new

- **Complete TypeScript ripmail port** — All mail processing now runs in-process with a new TypeScript module, eliminating Rust subprocess dependencies and improving performance and reliability.
- **Inbox and draft enhancements** — Added Haiku-backed draft refinement, collaborator tools, and richer notification prompts to streamline email workflows.
- **Cross-brain collaboration** — Introduced email-first cross-brain communication via the ask_collaborator tool, replacing legacy brain query tooling.
- **Tenant SQLite support** — Persist chat sessions, messages, and notifications per tenant in SQLite databases, improving data isolation and sync.
- **Phased i18n rollout** — Migrated multiple UI components and strings to i18next localization framework, enhancing multilingual support.
- **Onboarding and auth improvements** — Modular onboarding routes, tunnel host authentication, and a new AppShell provide a smoother user experience.

Also in this release: bug fixes for Google OAuth token revocation, UI padding and wrapping improvements in email iframes, Docker and CI simplifications, and updated documentation for architecture and deployment.

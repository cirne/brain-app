# Braintunnel release notes
**Release date:** 2026-05-12
**Changes since:** deploy-20260511-201427utc

This release brings a fresh new look to the inbox and chat UI, enhanced email message handling, and improved design documentation for a smoother development experience.

---

## What's new

- **Refreshed UI palette and typography** — We introduced a warm green color scheme and updated fonts to IBM Plex Sans and IBM Plex Mono for better readability and style consistency.
- **Inbox enhancements with HTML email support** — The ripmail schema now stores full HTML content, allowing richer email rendering in the inbox.
- **Improved chat experience** — Chat composer input now aligns perfectly with the transcript pane for a cleaner interface.
- **Design system improvements and documentation** — Added DESIGN.md spec and linting tools to keep design tokens consistent and maintainable.

Also in this release: minor backlog hygiene updates, archive of completed opportunities, and various bug fixes to polish the user experience.

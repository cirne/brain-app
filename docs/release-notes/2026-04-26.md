# Braintunnel release notes

**Release date:** April 26, 2026  
**Changes since:** April 24, 2026, 2:00 PM (previous release baseline)

This is the first file in this folder; later notes will point at the prior release’s date the same way.

---

## What’s new

- **Inbox triage (`/inbox-triage`)** — New guided mode to **clear noise fast, then work what matters**: the assistant loads your mail, proposes batch rules and safe bulk archive for obvious clutter, then walks what’s left with **one-tap next steps** instead of open-ended back-and-forth. Archived mail stays indexed and searchable—nothing is “gone.”
- **Chat & voice** — Listen to assistant replies with text-to-speech, use hold-to-speak for input, and get a smoother chat experience when picking up existing conversations (press-to-talk remains off until that flow is ready). Product feedback sent from chat can show a **draft preview** before submission.
- **Wiki & mail** — Easier wiki navigation from the Hub, clearer guardrails for how the agent can edit top-level wiki pages, and a better inbox reading experience (including improved rules behavior where it shows up in mail).
- **Phones & small screens** — Chat and the Hub are easier to use in a slide-over layout, with a cleaner top bar (your **@handle** moves into the Hub on narrow layouts), and broader mobile layout and touch improvements.
- **Getting started** — A clearer transition after you accept your profile, before you fully enter the app (including hosted demo tenant behavior).

**Also in this release:** Assorted bug fixes (for example hear-replies and quick-reply timing), wiki and agent-context polish, small inbox and Hub tweaks, better observability for hosted deployments, plus minor UI and infrastructure work behind the scenes.

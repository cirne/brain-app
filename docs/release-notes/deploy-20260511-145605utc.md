# Braintunnel release notes

**Release date:** 2026-05-11

**Changes since:** deploy-20260511-140802utc

This release introduces calendar synchronization enhancements and improved observability for our pipeline processes.

---

## What's new

- **Google Calendar sync** — Added a new sync module to refresh pipeline, enabling calendar-aware behavior in ripmail sources and agent tools.
- **Improved pipeline observability** — Staging Log queries now use entity GUIDs and support CLI backtick escaping for easier debugging.

Also in this release: minor configuration hooks and test improvements to support the new calendar sync features.

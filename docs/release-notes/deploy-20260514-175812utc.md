# Braintunnel release notes
**Release date:** 2026-05-14
**Changes since:** deploy-20260513-193058utc

This release brings a major update to the tunnels experience, unifying the activity surface and enhancing B2B capabilities with faster LLM-powered preflight checks. We also introduce Google Drive browsing in the hub and polish the UI for better alignment and navigation.

---

## What's new

- **Unified Tunnels Activity Surface** — Replaced multiple rails with a single Connections surface organized by person, combining email and tunnel activity for a streamlined view.
- **B2B Tunnel Preflight & Policy Evaluations** — Added fast LLM support for preflight checks, new CLI tools, updated UI labels, and inbound grant history recall for B2B agents.
- **Google Drive Hub Integration** — Enabled browsing of Google Drive folders within the hub, improving file selection workflows.
- **Tunnels Sidebar & Composer Enhancements** — Added tunnels list/detail UX with segmented controls, timeline APIs, and integrated draft restore in the chat composer.
- **Routing & Navigation Improvements** — Canonicalized tunnel URLs, simplified shell components, and updated mobile nav titles for clarity.

Also in this release: alignment fixes for tunnel pending review cards, updated tests reflecting UI changes, and refreshed documentation for opportunities and tunnel connections.

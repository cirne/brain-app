# Braintunnel release notes

**Release date:** 2026-05-07

**Changes since:** deploy-20260507-195933utc

This release focuses on enhancing local development parity with production, improving wiki header integration, and adding optional client-side source maps for better debugging.

---

## What's new

- **Local development parity** — start the local production server easily with `start-local.mjs`, aligning local and prod environments.
- **Wiki header improvements** — simplified wiring for the wiki-primary header to streamline UI consistency.
- **Optional client source maps** — enable client-side source maps via the `BRAIN_CLIENT_SOURCEMAP` flag for enhanced debugging.

Also in this release: updated documentation with new environment variable examples and added deploy release notes for better tracking.

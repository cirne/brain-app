# Braintunnel release notes

**Release date:** 2026-05-15

**Changes since:** deploy-20260515-175035utc

This release brings improved observability and reliability to mail synchronization, alongside UI polish and bug fixes for a smoother user experience.

---

## What's new

- **Enhanced observability** — Structured logging for Hub mail sync and ripmail sync failures improves monitoring and troubleshooting.
- **Improved backfill and refresh handling** — Fire-and-forget backfill/refresh jobs now route through brainLogger with job correlation and duration tracking.
- **Ripmail fixes and polish** — Skipping IMAP sync for Enron eval fixtures to avoid unnecessary operations and updated Tunnels copy for clarity.
- **Gmail sync improvements** — Narrowed refresh window without historyId, surfaced IMAP mailbox lastUid and backfill flags in hub status.
- **Hub and settings UI updates** — Simplified connections list, removed activity overview, and updated internationalization for better usability.
- **Bug tracking enhancements** — Added observability for ripmail refresh/backfill reliability gaps, bumping the unified bug ID to BUG-058.

Also in this release: minor fixes and polish to improve overall stability and user experience.

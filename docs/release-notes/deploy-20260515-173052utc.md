# Braintunnel release notes

**Release date:** 2026-05-15

**Changes since:** deploy-20260515-171323utc

This release includes a key fix improving maildir rebuild stability when handling attachments with unusual MIME types.

---

## What's new

- **Improved maildir rebuild robustness** — fixed a crash caused by non-string MIME types in attachments by falling back to extension-based MIME detection and enforcing type/subtype structure for strings.

Also in this release: minor polish and deployment process improvements.

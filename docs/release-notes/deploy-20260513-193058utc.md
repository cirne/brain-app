# Braintunnel release notes

**Release date:** 2026-05-13

**Changes since:** deploy-20260512-155651utc

This release introduces major enhancements to Braintunnel's chat and B2B collaboration features, alongside improvements in email handling and evaluation tooling. We've focused on streamlining the user experience for peer-to-peer chat, review workflows, and cold-query support, while also bolstering security and developer tooling.

---

## What's new

- **B2B Tunnel Chat and Collaboration** — Expanded chat routes and UI for B2B tunnels, including inbound approvals, peer mentions, and shared chat history with cold-query rate limits.
- **Review Queue and Wiki Integration** — Added a review queue UI with peer-review states and the ability to save tunnel conversations directly to the wiki.
- **Cold-Query and Workspace Search** — Introduced cold-query targets by email or user ID and enhanced workspace directory search with ranked sorting by display name and email.
- **Chat UX Improvements** — Inline tool indicators, compact tool blocks, streaming indicators for tunnels, and refreshed empty-chat notifications via SSE for real-time updates.
- **Email Compose and Draft Tools** — Added delete draft tool, thread-quoted reply bodies, and robust ripmail send/draft/search/filter support with agent integration.
- **Evaluation Suites and Testing** — New JSONL eval suites for mail-compose and wiki-kean tasks, plus Playwright helpers for B2B sharing and Enron demo workflows.
- **Security and CI Enhancements** — NPM malware lockfile checks integrated into CI, and gating installs with a shared denylist.

Also in this release: documentation updates for B2B chat architecture and OPPORTUNITIES specs, polish on agent tool icons, and minor bug fixes across chat and email components.

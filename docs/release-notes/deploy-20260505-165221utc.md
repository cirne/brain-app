# Braintunnel release notes
**Release date:** 2026-05-05
**Changes since:** deploy-20260504-111811utc

This release brings improved audio controls, enhanced wiki browsing and tool provenance, and a streamlined routing system for a smoother user experience.

---

## What's new

- **Dynamic audio conversation controls** — The mobile overflow menu now updates in real time with audio conversation state, allowing you to toggle audio easily with new Volume icons.
- **Improved routing architecture** — Hub, settings, and wiki surfaces now use a unified Route.zone system for cleaner navigation and URL handling.
- **Wiki tool enhancements and bug fixes** — Wiki tool outputs now include provenance annotations; browsing paths are unified and errors sanitized for clearer feedback.
- **UI improvements for mobile and editor** — Added BottomSheet and AnchoredActionMenu components, updated markdown editor, and improved mobile titles and overlay navigation.

Also in this release: flaky audio toggle tests fixed, local settings gitignored, and various bug reports filed and documented.

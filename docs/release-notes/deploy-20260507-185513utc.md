# Braintunnel release notes
**Release date:** 2026-05-07
**Changes since:** deploy-20260507-182957utc

This release focuses on improving the reliability and user experience of the wiki and vault landing pages, ensuring smoother loading and navigation.

---

## What's new

- **Improved wiki loading stability** — Now tracks loading generations to prevent stale fetches from causing stuck loading states.
- **Enhanced vault landing experience** — Clears selections and content appropriately when navigating to routes without a file path.
- **Robust page load success tracking** — Sets a clear pageLoadedOk flag on successful loads to improve UI responsiveness.
- **Simplified openFile logic** — Removed redundant branching to reduce effect complexity and potential bugs.
- **New regression tests** — Added coverage for aborted page GET requests and empty initial path scenarios to prevent regressions.

Also in this release: Documentation updated to track BUG-046 related to the unified wiki read route for better issue transparency.

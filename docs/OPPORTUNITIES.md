# Opportunities

Improvement ideas and future features discovered through usage. Each entry captures the problem, motivation, and proposed direction. **Product roots:** the wiki is the **compounding-LLM-wiki** half of the idea ([full text: karpathy-llm-wiki-post.md](karpathy-llm-wiki-post.md)); [ripmail](../ripmail/README.md) is the **indexed-mail** half — [VISION.md](VISION.md) ties them together.

Active and future work lives in [opportunities/](opportunities/). Shipped and deprioritized work moves to [opportunities/archive/](opportunities/archive/).

### Ripmail corpus (CLI + SQLite)

Indexed-mail and related **ripmail** opportunity specs live in **this** tree as **OPP-077+** (merged from former `ripmail/docs/opportunities/` numbering on 2026-05-01). `**[ripmail/docs/OPPORTUNITIES.md](../ripmail/docs/OPPORTUNITIES.md)`** is a pointer plus **historical stubs** where old `OPP-*` filenames still redirect.

**Numeric ids in *this* file are globally unique.** **[OPP-043](opportunities/OPP-043-google-oauth-app-verification-milestones.md)** is only the Google verification milestone plan. **Token metering** is **[OPP-072](opportunities/OPP-072-llm-usage-token-metering.md)**; **New Relic / usage CLI telemetry** is **[OPP-071](opportunities/OPP-071-llm-telemetry-traces-and-usage-cli.md)**. **Intent skill auto-loading** (`load_skill`) shipped as archived **[OPP-073](opportunities/archive/OPP-073-intent-based-skill-auto-loading.md)** (was a mistaken duplicate **OPP-035** filename). **Press-to-talk dev guardrail** history is archived **[OPP-074](opportunities/archive/OPP-074-press-to-talk-dev-only-feature-flag.md)**. **OPP-050** here is **[hosted wiki backup](opportunities/OPP-050-hosted-wiki-backup.md)**; **Apple Mail localhost** indexing is **[OPP-086](opportunities/OPP-086-applemail-localhost-mailbox.md)** (ex–ripmail OPP-050).
